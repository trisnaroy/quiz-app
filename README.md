# JavaScript Quiz App

A beginner-friendly interactive Quiz Application built using **HTML, CSS, and JavaScript**.  
This project is created for practicing core JavaScript concepts like DOM manipulation, event handling, logic building, and dynamic UI updates.

---

## Features

- Multiple choice questions
- Real-time answer checking
- Score tracking system
- Next question navigation
- Option lock after selection
- Simple and clean UI design
